Youtube Auto Factory

The youtube_auto_factory project is an autonomous video content manufacturing system specifically designed to automatically create short, tiger-themed documentary videos. This system integrates artificial intelligence as a writing agent that acts like a walking encyclopedia to compose factual and educational scripts. Through external API integration, the system intelligently researches visual assets from Pexels to obtain cinematic wildlife video clips and generates natural voiceover narrations. The entire asset compositing process is automated to produce high-quality videos ready for upload to various social media platforms.

A crucial aspect of this system's architecture is the use of Ollama, which is run within a Docker environment. This technical decision was made to address the challenges of low-resource hardware specifications, particularly a 2017 MacBook Air with 3.73 GB of RAM. By running Ollama via Docker, memory usage and CPU workload can be isolated and managed more efficiently, allowing language models like DeepSeek-R1 to operate stably without interfering with the intensive video rendering process. This approach ensures that the content production process remains smooth even when performed on a device with limited resources.
